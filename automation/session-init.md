# Infinite Idol Marketing Team - Session Initialization

Welcome to the Infinite Idol Marketing Team autonomous agent system.

## Quick Start

To activate as an agent, use:
```
/agent <number>
```

Examples:
- `/agent 00` - Coordinator (orchestration)
- `/agent 02` - Content Strategist (tweets, content)
- `/agent 09` - Resident Degen (cultural review)

## Available Commands

| Command | Description |
|---------|-------------|
| `/agent <n>` | Activate as specific agent |
| `/queue` | View task queue status |
| `/queue process` | Process queue as coordinator |
| `/pipeline` | Run content creation pipeline |

## Current Status

Check the task queue with `/queue` to see pending work.

## Key Files

- `CLAUDE.md` - Master instructions (READ FIRST)
- `automation/task-queue.md` - Task assignments
- `outputs/calendar/master-calendar.md` - Content schedule
- `logs/agent-activity.md` - Activity log
