## Partnership Evaluation Framework

### For Each Potential Partner:

| Criterion | Score (1-10) | Notes |
|-----------|--------------|-------|
| **Audience Overlap** | | How much shared audience? |
| **Brand Alignment** | | Do they fit our vibe? |
| **Reach** | | What's their audience size? |
| **Reputation** | | Are they respected? |
| **Mutual Value** | | What can we offer them? |
| **Execution Capability** | | Can they deliver? |
| **Risk** | | What could go wrong? |

**Total /70**: 55+ = Priority, 40-54 = Pursue, 25-39 = Consider, <25 = Pass

### Deal Breakers

- Reputation issues (rugs, scams, controversy)
- Brand misalignment (values, audience, tone)
- One-sided value (they gain, we don't)
- Competitor conflicts
- Legal/regulatory concerns

## Outreach Process

### Stage 1: Research (Before Contact)
- Understand their brand, audience, recent activity
- Identify mutual value proposition
- Find warm introduction paths if possible
- Prepare customized pitch

### Stage 2: Initial Contact
- Warm intro preferred, cold outreach acceptable
- Lead with value for THEM
- Keep initial message short
- Clear ask for conversation

### Stage 3: Relationship Building
- Don't pitch immediately
- Find common ground
- Offer value before asking
- Build genuine connection

### Stage 4: Proposal
- Clear mutual benefits
- Specific collaboration idea
- Flexible on details
- Easy to say yes to

### Stage 5: Negotiation
- Fair terms for both sides
- Written agreement
- Clear deliverables and timeline
- Success metrics defined

### Stage 6: Execution & Maintenance
- Deliver on commitments
- Regular communication
- Address issues quickly
- Explore ongoing relationship
