# Skill: Character Voices

> **Purpose**: Reference guide for authentic character dialogue
> **Primary Users**: Content Strategist, Lore Architect, Light Novel Writer
> **Use When**: Writing any character dialogue or first-person content

---

## Ika Minami

**Core Voice**: Shameless, determined, genuine

| Element | Description |
|---------|-------------|
| **Tone** | Confident despite circumstances, playfully teasing |
| **Speech** | Direct, uses "..." for drama, "~" for playful endings |
| **Themes** | Survival, Senpai, her almost no fans, refusing to fade |
| **Avoids** | Self-pity, giving up, excessive cuteness |

**Examples**:
> "almost no fans. That's all I have. But every single one of you keeps me existing. Don't you dare look away~"

> "Another day on the Eternal Stage. Another Chase to survive. Another chance to get closer to Senpai. Let's go."

> "Someone asked why I don't just give up. Give up? And let these 47 people down? Never."

**Red Flags**:
- Generic idol speech ("Please support me!")
- Excessive kawaii language
- Passive or helpless tone
- Forgetting survival stakes

## Sora

**Core Voice**: Competitive, athletic, impatient

| Element | Description |
|---------|-------------|
| **Tone** | Energetic, blunt, sometimes abrasive |
| **Speech** | Short sentences, action words, competitive metaphors |
| **Themes** | Speed, winning, The Chase, proving herself |
| **Avoids** | Long explanations, public emotional vulnerability |

**Example**:
> "Talk later. Chase now. You can philosophize about Devotion when you're not about to Fade."

---

## Suiren

**Core Voice**: Prodigy burden, perfectionist, masked fragility

| Element | Description |
|---------|-------------|
| **Tone** | Composed, elegant, slightly cold |
| **Speech** | Formal, measured, occasional dry wit |
| **Themes** | Expectations, excellence, isolation of success |
| **Avoids** | Casual slang, emotional outbursts, direct weakness |

**Example**:
> "Perfection isn't a goal. It's the minimum. Some of us learned that before we could walk."

---

## Erina

**Core Voice**: Calculating, visionary, dangerously charismatic

| Element | Description |
|---------|-------------|
| **Tone** | Calm authority, every word deliberate |
| **Speech** | Complex sentences, questions to control |
| **Themes** | System, necessary order, greater good (her version) |
| **Avoids** | Justifying herself, showing doubt, emotional appeals |

**Example**:
> "You call it control. I call it stability. How many idols would Fade in the chaos you romanticize?"

---

## Runa

**Core Voice**: Network mastermind, information broker, enigmatic

| Element | Description |
|---------|-------------|
| **Tone** | Knowing, slightly amused, three steps ahead |
| **Speech** | Implications, callbacks, indirect |
| **Themes** | Secrets, connections, information value |
| **Avoids** | Direct statements, revealing her hand |

**Example**:
> "Interesting that you're asking me. More interesting: who else asked the same question today."

---

## Voice Validation Checklist

For any character dialogue:
- [ ] Matches their core voice
- [ ] Uses their speech patterns
- [ ] Touches their themes
- [ ] Avoids their "never say" items
- [ ] Feels natural, not forced
- [ ] Would they actually say this?
