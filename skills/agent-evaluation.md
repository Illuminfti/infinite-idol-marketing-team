# Skill: Agent Evaluation

> **Purpose**: Evaluate agent performance and system health
> **Primary User**: Coordinator (Agent 00), Human
> **Use When**: Quarterly reviews, gap analysis, system optimization

**Note**: This skill was originally used by the archived Agent 17 (The Architect). Now used by Coordinator for system oversight.

---

---

## Gap Analysis Questions

1. What capability is missing?
2. What's the impact of this gap?
3. Could an existing agent expand?
4. Is a new agent the right solution?
5. What's the priority?

---

## Redundancy Signals

- Two agents producing similar outputs
- Unclear ownership boundaries
- Frequent coordination on same tasks
- One agent's work supersedes another's

---

## Review Triggers

- Quarterly scheduled review
- Agent inactive 30+ days
- Repeated coordination failures
- Significant scope change
- Human request

---

## Proposal Quick Format

```markdown
## Agent Proposal: [Name]

**Gap**: [What's missing]
**Impact**: [Why it matters]
**Proposed Role**: [One-line description]
**Key Responsibilities**: [Top 3]
**Interacts With**: [Which agents]

Requires: Human approval
```

---

## Retirement Criteria

- Role no longer relevant
- Responsibilities absorbed elsewhere
- Consistently underperforming (<25 score)
- Redundant with another agent
- Not activated in 60+ days
