# Skill: Templates

> **Purpose**: Standardized formats for agent outputs
> **All Agents**: Use these for consistency
> **Copy-paste ready**: Just fill in the brackets

---

## Content Submissions

### Tweet Submission
```markdown
## Tweet Submission

**ID:** TWEET-YYYY-MM-DD-XX
**Pillar:** [Ika Voice/Lore Drop/Founder Hype/Community]
**Scheduled:** [Date] [Time] JST
**Status:** Draft

### Content
[Tweet text - max 240 characters]

### Hashtags
#InfiniteIdol #SUI [additional tags]

### Media
[None / Image description / Asset link]

### Notes
[Context for reviewers]
```

### Thread Submission
```markdown
## Thread Submission

**ID:** THREAD-YYYY-MM-DD-XX
**Topic:** [Brief description]
**Pillar:** [Primary pillar]
**Scheduled:** [Date] [Time] JST
**Status:** Draft

### Tweet 1 (Hook)
[Content]

### Tweet 2
[Content]

[... additional tweets ...]

### Final Tweet (CTA)
[Content]

### Goal
[What this thread accomplishes]
```

```markdown
**Content:** [Reference ID or description]
**Date:** [Date]
**Reviewer:** [Agent name]

[APPROVED / REVISION REQUIRED / REJECTED]

- [ ] Character voice authenticity
- [ ] World logic consistency
- [ ] Timeline alignment

### Issues (if any)
1. [Issue]: [Location] - [Problem] - [Fix suggestion]

### Notes
[Additional context]
```

## Logging

### Activity Log Entry
```markdown
## [Date] - [Agent Name]

**Session:** [Start time] - [End time]
**Focus:** [Primary task]

### Actions
- [Action 1]
- [Action 2]
- [Action 3]

### Outputs
- [Output 1]: [Status]
- [Output 2]: [Status]

### Next Steps
- [What needs to happen next]

### Escalations
- [None / List of escalations filed]
```

### Decision Log Entry
```markdown
## Decision: [Brief Title]

**Date:** [Date]
**Decision Maker:** [Agent / Human-approved]
**Precedent:** [Yes/No]

### Context
[What required a decision]

### Decision
[What was decided]

### Rationale
[Why - cite sources/hierarchy]

### Impact
[What changes as result]
```
